## **Library System – Projeto de Estudo em Java (POO)**

Este é um pequeno sistema de gerenciamento de biblioteca desenvolvido em Java, com foco em praticar os conceitos fundamentais de Programação Orientada a Objetos.
O projeto simula ações simples como:
Adicionar livros
Listar livros
Emprestar livros
Devolver livros
É um projeto ideal para quem está iniciando em Java e deseja aplicar na prática classes, objetos, atributos, métodos e coleções.



## **Tecnologias utilizadas**

Tecnologias utilizadas

Java 17+
Paradigma de Programação Orientada a Objetos (POO)
java.util.List / ArrayList


## **Conceitos de POO praticados neste projeto**

Classes e Objetos
Encapsulamento
Responsabilidades de classes (SRP)
Construtores
Getters e Setters
ToString()
Lista de objetos (List<Book>)
Manipulação de estado (disponível/emprestado)
